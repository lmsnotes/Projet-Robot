### Projet-Robot

This repository features all files from a work completed in June 2025 as part of the *Projet Robot* course at Mines Saint-Étienne.

The objective of this project was to design and implement an embedded control system on an STM32 microcontroller allowing a mobile robot to be remotely controlled via Bluetooth communication using the USART protocol.

The project involved:
- Low-level embedded programming in C on STM32
- Configuration and use of USART peripherals for wireless communication
- Integration of a Bluetooth module for real-time remote control
- Motor control and command interpretation
- Debugging and testing of embedded systems in real conditions

The final system enabled reliable bidirectional communication between the robot and a remote device, allowing smooth navigation and command execution in real time.

This repository includes the source code, hardware configuration files, and the thoughts process developed throughout this project.
